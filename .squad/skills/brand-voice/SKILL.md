## Context
Every piece of content on IA Field Notes must follow strict brand voice rules. This skill mirrors the rules in `skills/SKILL-brand.md` for quick reference by all agents.

## Patterns

### Voice characteristics
- Practitioner-to-practitioner tone
- Direct, clear, honest about uncertainty
- Conversational but professional
- Field-credible — write from experience

### Banned words
Never use: revolutionary, unleash, game-changing, transformative (filler), cutting-edge, synergy, leverage (verb), unlock (overused), disrupt/disruptive, paradigm shift, best-in-class, next-generation

### Quick substitutions
- leverage → use
- unlock value → create value
- game-changing → significant
- cutting-edge → current, modern
- revolutionary → notable, important

### Citation rules
- External claims need linked sources
- Data points need source + date
- "According to..." must link to original

### Formatting
- Sentence case headings
- Oxford comma
- Spell out 1-9, numerals for 10+
- Em dashes with spaces

## Anti-Patterns
- No clickbait headlines
- No listicle framing ("7 Ways AI Will...")
- No press release tone
- No unsubstantiated productivity claims
- No "AI will replace X" — use "AI augments X"
